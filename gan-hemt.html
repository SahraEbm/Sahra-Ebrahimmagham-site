GaN HEMT Compact Model (Verilog-A)

Overview

This project presents a compact model for GaN High Electron Mobility Transistors (HEMTs) implemented in Verilog-A for circuit simulation environments. The model focuses on capturing key physical effects relevant to wide-bandgap semiconductor operation.

Repository

GitHub Project: GaN HEMT Compact Model Repository

Technical Focus

  • Drain current modeling
  • Channel charge behavior
  • Nonlinear device effects
  • High-frequency RF response considerations
  • SPICE-compatible Verilog-A implementation

Purpose

The goal of this work is to enable accurate circuit-level simulation of GaN HEMT devices while preserving key device physics in a computationally efficient compact form.
